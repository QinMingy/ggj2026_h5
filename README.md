# 黑心面具商人模拟器 (Sinister Mask Merchant Simulator)

一款融合社交推理、工艺制作与道德困境的东方奇幻模拟经营游戏。

## 🎮 游戏简介

扮演一位游走于灰色地带的面具商人，通过对话识破顾客的隐藏弱点，制作表面符合需求但暗藏克制元素的面具，从而在顾客遭遇不幸后获取额外收益。

## ✨ 核心特色

- **五行生克系统**：面具属性基于金木水火土的相生相克关系
- **社交推理**：通过对话线索和细节观察破译顾客弱点
- **道德困境**：在满足顾客与谋取暴利之间的心理博弈
- **东方美学**：融合中国戏曲面具、阴阳五行的独特画风

## 🎯 核心玩法

### 玩法循环
1. **顾客上门** → 查看顾客信息和需求
2. **对话交流** → 通过对话选项收集线索
3. **识别弱点** → 推理顾客的五行弱点
4. **制作面具** → 添加主属性满足需求，暗藏克制元素
5. **交付检验** → 通过伪装度判定
6. **获得金钱** → 成功交易获得收益
7. **查看战报** → 获取遗物和额外收益

### 五行相克关系
```
金 → 木 → 土 → 水 → 火 → 金
(克) (克) (克) (克) (克)
```

## 🎨 游戏系统

### 1. 对话系统
- 与顾客进行3-5轮对话
- 通过询问、观察收集线索
- 信任度影响对话效果
- 推理顾客的五行弱点

### 2. 制作系统
- 选择6种基底面具之一
- 添加纹样、颜料、符文等材料
- 满足顾客的属性需求
- 暗藏克制元素（可选）
- 控制显眼度避免被识破

### 3. 交付系统
- 顾客检验面具
- 显眼度影响识破概率：
  - <30%：安全
  - 30-50%：警惕（10%识破）
  - 50-80%：危险（40%识破）
  - >80%：必定识破

### 4. 战报系统
- 完成3单后触发
- 含克制元素的面具会导致顾客遭遇不幸
- 获得遗物和额外金钱
- 影响声望

## 🎭 顾客类型

游戏包含5种顾客类型：

1. **剑客** - 需要金属性面具，弱点：木
2. **书生** - 需要木属性面具，弱点：火
3. **商人** - 需要土属性面具，弱点：木
4. **猎人** - 需要水属性面具，弱点：土
5. **道士** - 需要火属性面具，弱点：水

## 🛠️ 材料系统

### 基底面具
- 全脸面具、鬼面、半脸面具、眼罩、兽面、戏曲面具

### 材料类型
- **纹样**：龙纹、凤纹、云纹、兽纹、山纹
- **颜料**：青色、红色、蓝色、银色、黄色
- **符文**：刚猛符、生机符、流转符、炎爆符、厚土符

每种材料都有对应的五行属性、增益值和显眼度。

## 📊 游戏数值

- **初始金钱**：500两
- **面具售价**：80-150两
- **遗物收益**：50-100两 + 额外材料
- **声望系统**：影响信誉和顾客类型

## 🚀 如何开始

1. 打开 `index.html` 文件即可开始游戏
2. 点击"开始游戏"进入商店
3. 点击"下一位顾客"开始接待
4. 通过对话收集线索，推理弱点
5. 制作面具并交付
6. 查看战报获取额外收益

## 💡 游戏策略

### 安全策略
- 只满足顾客需求，不添加克制元素
- 获得稳定收益，保持良好信誉

### 暴利策略
- 识别顾客弱点，暗藏克制元素
- 控制显眼度在30%以下
- 等待战报获取高额遗物

### 平衡策略
- 部分订单添加克制元素
- 在收益和风险之间取得平衡
